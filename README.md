# React Redux Vite Todo App

This is a todo application built using React with Redux and Vite as the build tool.

## Setup

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-url>
   cd react-redux-vite-todo-app
   
## Install dependencies
   npm install

## Running the application
  To run the application locally
  npm run dev

## Usage
Add todos using the input field.
Edit todos by clicking the edit button.
Delete todos by clicking the delete button.
Mark todos as completed by checking the checkbox.

## Deployment
For deployment options, consider using platforms like Vercel or Netlify.
